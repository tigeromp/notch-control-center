## Reddit — r/macapps

**Title:**
```
[Open Source] Notch Control Center — live stocks, sports, music & widgets in your MacBook notch (Swift/SwiftUI)
```

**Body:**
```
I built a native macOS menu bar app that turns the notch into an expandable control panel.

**What it does:**
- Music controls (Apple Music + Spotify)
- Live stock ticker with green/red flash on price changes
- Live sports scores only (NFL, NBA, MLB, NHL, soccer, etc.)
- Weather, news, calendar, timer, crypto
- Three-finger swipe down to open
- Customizable appearance (accent color, text size, panel style)

**Tech:** Swift + SwiftUI, no Electron.

**Install:** Download the zip from GitHub Releases, or build from source.

GitHub: https://github.com/tigeromp/notch-control-center

Free & open source (MIT). Would love feedback — especially on gestures and which widgets you'd want next.

**Permissions:** Accessibility (for 3-finger open), Automation (Spotify), Calendar (optional).
```
